A Filosofia da Legibilidade

  • O código é lido mais vezes do que escrito.
  • Sem símbolos complexos como {} ou ; para blocos.
  • Utiliza indentação obrigatória para estruturar lógica.
  • Foco na clareza e soluções diretas e óbvias.

Por que Indentação?

Quando você escreve um programa, está redigindo um documento para outros humanos. O Python força um visual limpo, eliminando ruídos sintáticos para destacar a lógica.

Tipos Básicos de Dados

Inteiros (int)

Números inteiros, positivos ou negativos, sem decimais (ex: 10, -5, 0).

Ponto Flutuante (float)

Números reais com parte decimal. Ex: 3.14, 2.0.

Strings (str)

Sequências imutáveis de caracteres manipuladas como texto. Ex: 'Olá'.

Booleanos (bool)

Tipo lógico que assume True ou False. Base do controle de fluxo.

Estruturas de Dados Nativas

Listas (list)

Coleções ordenadas e mutáveis. Usa [ ].

Tuplas (tuple)

Coleções ordenadas e imutáveis. Usa ( ).

Dicionários (dict)

Pares chave-valor não ordenados. Usa { }.

Conjuntos (set)

Elementos únicos não ordenados. Usa { }.

Listas vs Tuplas

Listas

  • Sintaxe: [1, 2, 3]
  • Mutável: pode ser alterada.
  • Ideal para coleções dinâmicas.

Tuplas

  • Sintaxe: (1, 2, 3)
  • Imutável: não muda após criada.
  • Ideal para coordenadas ou chaves.

Explorando Dados Avançados

São coleções de pares chave-valor. Funcionam como um dicionário real: você busca uma palavra (chave) para encontrar sua definição (valor). Muito rápidos para recuperação de dados.

Coleções de elementos únicos. Eliminam duplicatas automaticamente e são ideais para operações matemáticas como união e interseção de conjuntos.

Controle de Fluxo Condicional

if (Condição Verdadeira)
elif (Outra Condição)
else (Nenhuma Anterior)

Estrutura Condicional na Prática 

# Exemplo de verificação
idade = 18

if idade >= 18:
    print("Maior de idade")
elif idade > 0:
    print("Menor de idade")
else:
    print("Idade inválida")

Laços de Repetição (Loops)

Laço FOR

Itera sobre sequências (listas, strings). "Para cada item nesta sequência, execute isto".

Laço WHILE

Executa enquanto uma condição booleana permanecer True. Requer atenção à condição de parada.

Funções: O Coração da Modularidade

Em vez de escrever comandos repetidos, encapsulamos lógicas em funções com a palavra-chave def .

  • Promove reutilização de código.
  • Facilita manutenção e testes.
  • Podem receber argumentos e usar return .

Alerta: Loop Infinito

Ao usar o laço while , certifique-se de que a condição se tornará False em algum momento. Caso contrário, o programa travará executando a mesma tarefa para sempre!
